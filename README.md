# Tienda Online en Python con SQLite

Aplicación de consola desarrollada en Python que simula una tienda online.
Permite registrar usuarios, mostrar productos, realizar pedidos y consultar el historial,
almacenando la información en una base de datos SQLite.

## Tecnologías
- Python 3
- SQLite (sqlite3)

## Ejecución
```bash
python3 tienda.py
